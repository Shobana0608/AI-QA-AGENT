Setup Guide
Up & Running in 4 Steps
01

Install Ollama

Download from ollama.com and pull the required models.

ollama pull qwen2.5
ollama pull nomic-embed-text
02

Install Langflow

Create a clean Python 3.11 environment and install Langflow.

conda create -n qaagent python=3.11 -y
conda activate qaagent
pip install langflow
python -m langflow run
03

Wire the Flow

Open localhost:7860 → New Flow → Connect Chat Input → Agent (Ollama / qwen2.5:latest) → Chat Output. Enable Tool Mode ON.

04

Add QA Prompt & Test

Paste the Senior QA Engineer system prompt into Agent Instructions → click Playground → start testing!
